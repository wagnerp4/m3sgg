Core Components
---------------

The M3SGG project consists of several key components:

* **Datasets Module**: Handles dataset loading and preprocessing
* **Core Library**: Configuration, models, detectors, evaluation, and training
* **Models**: Implementations including STTran, SceneLLM, Tempura, STKET, OED, VLM
* **CLI Module**: Command-line interface and utilities
* **Language Module**: Summarization and evaluation tools
* **Utils Module**: Helper functions and utilities
